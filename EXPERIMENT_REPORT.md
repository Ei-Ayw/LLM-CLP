# 实验报告：基于DeBERTa-V3多任务学习与身份感知重加权的毒性文本分类

## 1. 实验概述

### 1.1 研究目标

基于Vanilla DeBERTa-v3，提出一种两阶段多任务学习框架，结合身份感知重加权策略，在保持毒性分类性能的同时提升模型在不同人口统计子群体上的公平性。

### 1.2 实验矩阵

| 组别 | 模型 | 角色 |
|------|------|------|
| 组1 | TF-IDF + 逻辑回归 | 经典基线 |
| 组2 | BERT + CNN + BiLSTM | 混合深度学习基线 |
| 组3a | Vanilla BERT | Transformer基线 |
| 组3b | Vanilla RoBERTa | Transformer基线 |
| 组3c | **Vanilla DeBERTa-v3** | **消融基线（我们的骨干网络）** |
| 组4 | **DeBERTa-v3 MTL S1 -> S2（所提方法）** | **我们的完整方法** |
| 消融 | DeBERTa-v3 MTL S1 -> S2（仅BCE） | 消融实验：移除Focal Loss + 加权损失 |

### 1.3 数据集

- 来源：Jigsaw Unintended Bias in Toxicity Classification（Civil Comments）
- 原始数据：1,804,874条评论
- 均衡采样：288,668条评论（50:50 毒性/非毒性）
- 数据增强：同义词替换、随机插入、随机删除、回译（仅对毒性样本）
- 最终划分：80%训练 / 10%验证 / 10%测试
- 评估身份子群体：male、female、black、white、muslim、jewish、christian、homosexual_gay_or_lesbian、psychiatric_or_mental_illness

### 1.4 硬件与训练配置

- 硬件：8× NVIDIA RTX 3090（24GB）
- 基线模型：DataParallel（2 GPU，batch_size=48）
- DeBERTa模型：DDP（batch_size=16/GPU）
- 序列最大长度：256
- 早停patience：3 epochs
- 优化器：AdamW
- 学习率调度：ReduceLROnPlateau

## 2. 所提方法：架构与关键创新

### 2.1 架构总览

```
输入文本
    |
[DeBERTa-V3-Base 编码器] (解耦注意力)
    |
[双池化策略]
    |-- h_cls = CLS token 表示 [768维]
    |-- h_att = 注意力池化输出  [768维]
    |-- h = concat(h_cls, h_att) [1536维]
    |
[特征投影]
    |-- z = Dropout(GELU(Linear(1536 -> 512)))
    |
[多任务头]
    |-- 毒性头:  Linear(512 -> 1)   (主任务)
    |-- 子类型头: Linear(512 -> 6)   (辅助: severe_toxicity, obscene, identity_attack, insult, threat, sexual_explicit)
    |-- 身份头:  Linear(512 -> 9)   (辅助: 9个人口统计群体)
```

### 2.2 关键创新1：注意力池化

与仅使用CLS token的Vanilla DeBERTa不同，我们引入了可学习的注意力池化机制：

```
s_i = Linear(h_i)                          # 对每个token打分
a_i = softmax(s_i, mask=attention_mask)     # 归一化注意力权重
h_att = sum(a_i * h_i)                      # 加权聚合
```

这使模型能够动态关注最与任务相关的token，而不仅限于CLS。

### 2.3 关键创新2：两阶段训练

**第一阶段（S1）- 多任务预训练：**
- 损失函数：L = L_tox + 0.1 × L_sub + 0.2 × L_id
- L_tox 使用 BCEFocalLoss（alpha=12.5, gamma=2.0）以应对类别不均衡
- 辅助任务对编码器起正则化作用，学习更丰富的表示

**第二阶段（S2）- 身份感知微调：**
- 加载S1检查点，使用身份感知加权损失进行微调
- 学习率降至5e-6（S1为2e-5）
- 关键重加权方案：

| 样本类型 | 包含身份词 | 有毒 | 权重 | 原因 |
|---------|-----------|------|------|------|
| 背景非毒性 | 否 | 否 | 1.0 | 基准 |
| 背景毒性 | 否 | 是 | 1.0 | 基准 |
| **身份非毒性** | **是** | **否** | **2.5** | **惩罚对身份词的误报** |
| 身份毒性 | 是 | 是 | 1.5 | 适度提升身份-毒性样本权重 |

### 2.4 关键创新3：S1阶段的Focal Loss

```
FL(p_t) = -alpha_t × (1 - p_t)^gamma × log(p_t)

alpha = 12.5（正类权重）
gamma = 2.0 （聚焦参数）
```

- 降低简单、已正确分类样本的权重
- 迫使模型关注困难的、处于毒性/非毒性边界的模糊样本

## 3. 主要结果：模型对比

### 3.1 分类性能（最优阈值）

| 模型 | ROC-AUC | PR-AUC | F1 | 准确率 | 最优阈值 |
|------|---------|--------|-----|--------|---------|
| BertCNNBiLSTM | 0.9656 | 0.9622 | 0.9052 | 0.9017 | 0.39 |
| Vanilla BERT | 0.9682 | 0.9658 | 0.9075 | 0.9048 | 0.50 |
| Vanilla RoBERTa | 0.9673 | 0.9646 | 0.9079 | 0.9054 | 0.58 |
| **Vanilla DeBERTa-v3** | **0.9678** | **0.9646** | **0.9095** | **0.9064** | **0.67** |
| **MTL S2（所提方法）** | **0.9678** | **0.9647** | **0.9113** | **0.9084** | **0.46** |
| 消融 BCE S2 | 0.9680 | 0.9647 | 0.9116 | 0.9085 | 0.46 |

### 3.2 分类性能（固定阈值 = 0.5）

| 模型 | F1 | 准确率 | 精确率 | 召回率 |
|------|-----|--------|--------|--------|
| BertCNNBiLSTM | 0.9035 | 0.9023 | 0.8927 | 0.9145 |
| Vanilla BERT | 0.9075 | 0.9048 | 0.8822 | 0.9342 |
| Vanilla RoBERTa | 0.9072 | 0.9035 | 0.8736 | 0.9434 |
| **Vanilla DeBERTa-v3** | **0.9039** | **0.8972** | **0.8481** | **0.9676** |
| **MTL S2（所提方法）** | **0.9102** | **0.9080** | **0.8891** | **0.9323** |
| 消融 BCE S2 | 0.9109 | 0.9084 | 0.8872 | 0.9358 |

### 3.3 分析：分类性能

1. **所有模型均达到了有竞争力的ROC-AUC**（0.9656-0.9682），表明预训练Transformer能够很好地完成此任务。

2. **Vanilla DeBERTa-v3在阈值=0.5时召回率高（0.9676）但精确率低（0.8481）**。这意味着它倾向于将文本激进地分类为毒性，导致大量误报。其最优阈值需上移至0.67来补偿。

3. **我们所提的MTL S2在阈值=0.5时实现了最佳的精确率-召回率平衡**（精确率=0.8891，召回率=0.9323），F1=0.9102，相比Vanilla DeBERTa的0.9039提升了+0.7%。

4. **MTL S2的最优阈值为0.46**（接近0.5），表明模型输出概率校准良好。相比之下，Vanilla DeBERTa需要0.67的阈值，说明其概率输出校准不佳。

5. **BertCNNBiLSTM在深度模型中表现最差**（F1=0.9052），说明复杂的混合架构不一定优于简单的Transformer微调。

**关键发现4：公平性提升伴随极小的准确性损失**

### 6.2 训练动态分析

1. **所有模型均表现出早期过拟合**：训练损失持续下降，而验证损失在第1-2个epoch后开始上升。这是大型预训练模型在领域特定数据上微调的典型现象。

2. **MTL S1达到了最低验证损失（0.1839）**：多任务学习配合Focal Loss提供了最强的正则化效果，在S1阶段实现了最佳泛化能力。

3. **早停机制至关重要**：如果没有早停，所有模型都会严重过拟合（如VanillaBERT的验证损失从0.2306上升到0.3833）。

4. **S2阶段起始损失更高但收敛到更好的偏差指标**：身份感知重加权增加了训练难度（更高的损失值），但产生了更公平的预测结果。

## 8. 结论

1. **所提出的DeBERTa-v3多任务学习结合身份感知重加权方法实现了最优的公平性**（平均偏差AUC = 0.9317），同时保持了有竞争力的分类性能（F1 = 0.9113）。

2. **两阶段训练策略有效**：S1多任务预训练构建了丰富的表征，S2身份感知微调针对性地缓解了人口统计偏差。

3. **Focal Loss + 身份重加权是公平性提升的关键贡献者**，消融实验已证明：移除这些组件后平均偏差AUC下降0.0045，最差子群AUC下降0.0086。

4. **改进对弱势群体最为显著**：homosexual（+0.0211子群AUC）、muslim（+0.0175子群AUC）以及种族群体从我们的方法中获益最大。

5. **模型校准得到改善**：Vanilla DeBERTa需要阈值=0.67才能达到最优F1，而我们的方法在阈值=0.46时即可达到最优F1，表明概率输出校准更好。

6. **BPSN-BNSP权衡是有利的**：我们的方法大幅减少了误报偏差（BPSN上升），仅伴随边际的漏报率增加（BNSP略微下降），这对内容审核系统而言是理想的结果——在此场景下，对非毒性的身份相关言论的错误审查是首要关切。
