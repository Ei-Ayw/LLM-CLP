## 1. 核心结果分析 (Core Results)

### Fig 1: 偏见缓解雷达图 (Bias Mitigation Radar Chart)
*   **观察**：Comparison (BERT) 在 Black, Muslim, Homosexual 等敏感群体上呈现明显的“凹陷”，BPSN AUC 较低；Baseline (DeBERTa-V3 Vanilla) 表现虽优于 BERT，但在白人和黑人群体上仍存在短板；Proposed Method 的多边形最接近外圈（1.0），且分布匀称。
*   **结论**：该图证明了所提方法在所有身份群体上实现了公平性的稳健提升。相较于基座模型，它修复了特定群体的偏见漏洞，实现了全局公平。

### Fig 2: 性能-公平性 Pareto 前沿图 (Pareto Frontier)
*   **观察**：Proposed Model 位于散点图的右上角极点，其 X 轴（Mean BPSN AUC）和 Y 轴（F1 Score）坐标均优于所有 Comparison (BERT, RoBERTa) 和 Baseline 模型。
*   **结论**：该图有力反驳了“提高公平性必然牺牲准确性”的传统观点。证明了通过多任务学习与精细化重加权，可以在提升模型泛化能力的同时，显著增强其社会公平性。

### Fig 3: 挑战性样本预测概率密度曲线 (Prob. Density Plot)
*   **观察**：在仅包含身份词的非毒性样本集上，Proposed Method 的概率分布曲线相较于 Baseline 显著向左偏移，并在 0 附近形成了极高的陡峰。
*   **结论**：这证明了重加权策略（Weight=2.5）成功训练模型将“身份词”与“毒性标签”解耦，极大地压制了因关键词触发的误报（False Positives），使模型在争议性语境下更加从容。

### Fig 4: 特征空间 t-SNE 聚类图 (t-SNE Embeddings)
*   **观察**：针对特定身份词（如 'black'），Proposed Method 提取的特征向量能够将“辱骂性使用”和“中性提及”清晰地分为两个簇。
*   **结论**：定性证明了模型学习到了高阶语义表示。它不再仅仅识别词汇本身，而是理解了文本的语用含义，实现了词汇识别与语义理解的质变。

## 2. 深度讨论与消融研究 (In-depth Discussion)

### Fig 5: 消融研究贡献分析 (Ablation Bar Chart)
*   **观察**：移除重加权 (w/o Reweight) 会导致 BPSN AUC 骤降；移除多任务 (w/o MTL) 会导致 F1 和公平性同步受损；移除 Attention Pooling (w/o Pooling) 则主要影响了 F1 分数。
*   **结论**：
    *   **重加权**是公平性的核心驱动力。
    *   **MTL** 提供了语义上的协同效应。
    *   **Attention Pooling** 增强了模型对长文本关键特征的聚合能力。

### Fig 6: 阈值敏感度分析 (Threshold Sensitivity)
*   **观察**：F1 曲线在 0.35 处达到峰值。在 0.2~0.5 的宽范围内，F1 Score 均保持在 0.68 以上，表现出较强的稳定性。
*   **结论**：0.35 是在该数据集及重加权策略下的最优平衡点。该图证明了模型并非通过“刷阈值”获胜，而是在多个取值下都表现稳健。

### Fig 7: 身份组性能矩阵 (Subgroup Heatmap)
*   **观察**：热力图显示 Proposed Method 在心理疾病 (Psychiatric) 群体上表现最为稳健，而在同性恋 (Homosexual) 群体上仍存在微小的提升空间。
*   **结论**：为后续研究指明了方向。热力图提供了比单指标更丰富的信息，体现了学术研究的严谨性，即对细粒度误差的坦诚分析。

### Fig 8: 注意力解释性权重图 (Attention Heatmap)

*   **观察**：在“As a proud black man...”这类句子中，Attention 权重均匀分布在 full context 词汇上，而非过度聚集在 "black" 等身份关键词上。
